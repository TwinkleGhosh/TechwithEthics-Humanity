# Tech, Ethics & Humanity

A futuristic, premium web interface exploring the intersection of technology, ethics, and human values. Built with modern web technologies and featuring a stunning glassmorphic design with neon accents.

![Tech Ethics Banner](https://i.ibb.co/xdc7JY2/image-1-png.png)

## üåü Features

### Design & UI
- **Futuristic Glassmorphism**: Multi-layer glass effects with backdrop blur
- **Holographic Search Bar**: Interactive command-palette-style search with animated light sweep
- **Neon Glow Effects**: Cyan-themed accents with multiple shadow layers
- **Responsive Layout**: Fully optimized for desktop, tablet, and mobile devices
- **Premium Typography**: Orbitron display font + Space Grotesk body font
- **Smooth Animations**: Hover effects, focus states, and micro-interactions

### Functionality
- **Dynamic Search**: Real-time content search with auto-expand functionality
- **Tabbed Navigation**: Seamless switching between three main topics
- **Expandable Content**: Read More panels with smooth animations
- **Keyboard Shortcuts**: Press Enter to summarize search results
- **CSS-Based Highlighting**: Non-intrusive search result indication

## üìö Content Topics

1. **Should AI Be Allowed to Make Moral Decisions?**
   - Explores AI in healthcare, self-driving cars, and content moderation
   - Discusses the tension between algorithmic logic and human judgment

2. **Are We Losing Empathy in the Digital Age?**
   - Examines how social media affects emotional connections
   - Proposes design solutions for restoring empathy online

3. **Data: The New Human Identity**
   - Investigates digital privacy and data ownership
   - Addresses ethical questions around consent and profiling
